## Project: Sales Data Analysis using SQL

# Project Overview

# Dataset
Sales dataset with customers, product, and product information

# What I Did
- Created relational database (customers, orders, products)
- Cleaned and transformed raw dataset
- Handled duplicate records
- Established relationships using foreign keys
- Performed data analysis using JOIN queries
  
# Key Insights
- Identification of high-value customers
- Customers with highest number of orders identified
- Products contributing most to revenue
- Some regions show higher sales concentration
- Some regions show higher oreder concentration
